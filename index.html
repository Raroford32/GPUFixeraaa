oobabooga/text-generation-webui Setup

Welcome to the setup page for oobabooga/text-generation-webui. Follow these steps to get started:

  1. Ensure that you have Python 3.8 or higher installed on your system.
  2. Open a terminal or command prompt in the directory containing the setup_and_run.py script.
  3. Run the following command to start the setup and launch the web UI:
python setup_and_run.py

The script will perform the following tasks:

  • Check and verify CUDA and NVIDIA driver installation
  • Install required Python packages
  • Clone the oobabooga/text-generation-webui repository
  • Configure the application to use all 8 A100 GPUs
  • Start the text generation web UI

Once the setup is complete and the web UI is running, you can access it at:

http://localhost:5000

If you encounter any issues during the setup process, please check the console output for error messages and troubleshooting information.
